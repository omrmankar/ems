INTRODUCTION
------------

This module redirects a user to a specific URL after logout.

 * For a full description of the module, visit the project page:
   https://www.drupal.org/project/redirect_after_logout

 * To submit bug reports and feature suggestions, or to track changes:
   https://www.drupal.org/project/issues/redirect_after_logout


INSTALLATION
------------

  * Place the Redirect After Logout module into your modules directory.
    This is normally the "sites/all/modules" directory.

  * Go to admin/build/modules. Enable the module.

  * The Redirect After Logout module is found in the "Other" section.

CONFIGURATION
-------------

  * Go to admin/config/system/redirect_after_logout
    and configure the module.
